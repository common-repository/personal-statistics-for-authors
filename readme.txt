Personnal Statistics for Authors use Facebook, Twitter and Google Analytics API to get the data of your posts.

== Description ==

Personnal Statistics for Authors is the plugin required for every blog. It uses the Facebook, Twitter and Google Analytics API to get the data of your posts (shares, tweets, keywords used to find it, visits from smartphones, bounce rate...). You don't need to switch between your Analytics dashboard and everything else : PSfA is here to do that job for you.

== Installation ==

1. Upload 'personnal-statistics-for-authors' to the '/wp-content/plugins/' directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to 'Personnal Statistics for Authors' under the 'Settings' tab and configure the plugin
